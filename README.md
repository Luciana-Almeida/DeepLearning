Repositório de anotações, códigos e projetos desenvolvidos durante a **Especialização em Deep Learning**.

## 📌 Sobre este repositório

Este repositório reúne todo o material produzido ao longo da especialização: implementações práticas, notebooks de estudo, experimentos com diferentes arquiteturas e projetos aplicados. O objetivo é documentar o aprendizado de forma organizada e progressiva.

          ## 📚 Conteúdo por módulo

          ### 01 · Fundamentos
          Conceitos matemáticos essenciais (álgebra linear, cálculo, probabilidade), gradiente descendente e fundamentos de redes neurais.

          ### 02 · Redes Neurais Artificiais
          Implementação do Perceptron, redes multicamadas (MLP), funções de ativação (ReLU, Sigmoid, Softmax) e regularização (Dropout, Batch Norm).

          ### 03 · Redes Neurais Convolucionais (CNN)
          Arquiteturas clássicas (LeNet, VGG, ResNet), transfer learning e aplicações em classificação e detecção de imagens.

          ### 04 · Redes Recorrentes (RNN / LSTM / GRU)
          Modelagem de sequências, análise de séries temporais e processamento de linguagem natural básico.

          ### 05 · Transformers e Atenção
          Mecanismo de atenção, arquitetura Transformer, fine-tuning de modelos pré-treinados (BERT, GPT).

          ### 06 · Modelos Generativos
          Redes Adversariais Generativas (GANs), Autoencoders Variacionais (VAEs) e geração de dados sintéticos.

          ### 07 · Projetos Práticos
          Aplicações completas integrando os conhecimentos dos módulos anteriores — desde o pré-processamento até a avaliação do modelo.

          ## 🚀 Como executar

          1. Clone o repositório:
          2.    ```bash
                   git clone https://github.com/Luciana-Almeida/DeepLearning.git
                   cd DeepLearning
                   ```

                2. Crie e ative um ambiente virtual:
                3.    ```bash
                         python -m venv venv
                         source venv/bin/activate  # Linux/macOS
                         venv\Scripts\activate     # Windows
                         ```

                      3. Instale as dependências:
                      4.    ```bash
                               pip install -r requirements.txt
                               ```

                            4. Abra os notebooks:
                            5.    ```bash
                                     jupyter notebook
                                     ```
