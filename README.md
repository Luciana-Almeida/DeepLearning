# 🧠 Deep Learning — CIN/UFPE

Repositório de códigos, exercícios e experimentos desenvolvidos durante a **Especialização em Deep Learning do CIn/UFPE**.

## Sobre

Este espaço reúne os materiais que produzo ao longo da especialização com o objetivo de fixar o aprendizado na prática. Os códigos acompanham os conteúdos vistos nas aulas e cobrem desde os fundamentos de redes neurais até arquiteturas mais avançadas.

## Tecnologias

- Python
- TensorFlow / Keras
- PyTorch
- NumPy, Pandas, Matplotlib
- Jupyter Notebook / Google Colab
